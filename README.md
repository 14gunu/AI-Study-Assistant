# 🎓 AI Study Assistant

## 📖 Project Overview

AI Study Assistant is an AI-powered learning application built using Streamlit and Google Gemini AI. The application helps students convert study materials into useful learning resources such as summaries, important questions, quizzes, and flashcards.

Students can upload PDFs, images, screenshots, or paste notes directly into the application and receive AI-generated study content instantly.

---

## 🎯 Problem Statement

Students often spend a significant amount of time creating summaries, revision questions, and flashcards from lengthy study materials.

This project automates that process using Artificial Intelligence, helping students study more efficiently and improve their learning experience.

---

## 🚀 Features

### 📂 Multi-Format Input

* Upload PDF files
* Upload Images and Screenshots
* Paste Notes Directly

### 🤖 AI-Powered Learning Tools

* 📝 Generate Summaries
* ❓ Generate Important Questions
* ✅ AI Quiz Mode
* 🃏 Generate Flashcards

### 🎨 User Interface Features

* Modern Dark Theme
* Difficulty Selection (Easy, Medium, Hard)
* Progress Tracker
* Study Streak Counter
* Download Generated Results

## ⚙️ Installation Guide

### Step 1: Clone Repository

```bash
git clone https://github.com/14gunu/AI-Study-Assistant.git
```

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run Application

```bash
streamlit run app.py
```

### Step 4: Open Browser

```text
http://localhost:8501
```

### Step 5: Enter Gemini API Key

Enter your Google Gemini API key and start using the application.
