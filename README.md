# AI Study Assistant

An AI-powered study assistant built using Streamlit and Google Gemini.

## Features

* Upload Images
* Upload PDFs
* Paste Notes
* Generate Summaries
* Generate Important Questions
* Generate MCQs
* Generate Flashcards

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* Pillow
* PyPDF

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run application

streamlit run app.py
